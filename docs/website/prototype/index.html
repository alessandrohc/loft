What is Loft?

Loft is a object persistence framework and an object manipulation toolkit written in the Lua language.

The primary purpose of Loft is to establish a standard interface for both accessing objects from their persistent repository and for writing repository providers. A repository can be a relational database, the network, a set of files on the filesystem, a directory server or any other form of data persistence.

Support for different types of repositories are implemented on persistence providers. Most of them can provide all you need to access objects in your application, but Loft allows for you to write your own provider and tweak some functionality for better performance.
